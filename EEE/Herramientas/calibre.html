Calibre

El calibre o pie de rey es una herramienta de medición que se usa para medir con mucha precisión el tamaño de objetos pequeños.

Sirve para medir:

  • Diámetros exteriores (por ejemplo, el grosor de una pieza).
  • Diámetros interiores (el hueco de un tubo).
  • Profundidades (la altura o profundidad de un agujero).

Tiene dos escalas:

  • Una escala fija (en milímetros o pulgadas).
  • Una escala móvil (el nonio), que permite leer las décimas y centésimas de milímetro.
